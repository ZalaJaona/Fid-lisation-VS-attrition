# Fid-lisation-VS-attrition
En construisant ce dashboard Power BI, je me suis intéressé à un enjeu clé :
 comment identifier les clients à risque pour protéger la créance relationnelle dans une banque ?
En analysant l’âge, le pays et le nombre de produits détenus, plusieurs signaux apparaissent :
Les clients multi‑produits sont paradoxalement les plus exposés
L’attrition varie fortement selon les pays
Certaines tranches d’âge montrent une sensibilité plus élevée
Comprendre ces dynamiques permet aux équipes métiers de :
 ✔️ anticiper les départs
 ✔️ renforcer la relation client
 ✔️ préserver la valeur créée dans le temps
C’est exactement ce que j’aime dans la data : donner du sens aux chiffres pour améliorer la relation client.
